# Claims

- Mechanism claim tested: affordance maps should represent alternate grasps, poses, supports, and semantic roles, not only currently observed affordances.
- Evidence claim: a deterministic manipulation-affordance benchmark with strong baselines, ablations, stress sweeps, uncertainty/probe comparisons, and negative cases was added.
- Supported local finding: counterfactual maps improve over static observed/graph/VLM affordance baselines and ablations are coherent.
- Unsupported main claim: the proposed method is not submission-ready because it does not decisively beat interactive affordance probing.
- Unsupported claim explicitly avoided: no claim of SOTA robot affordance learning, real robot deployment, or high-fidelity simulator validation.
